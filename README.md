# XEN TradeHub

XEN TradeHub is an independent asset management service provider focused on structured trading support, operational transparency, and disciplined market participation.

![Homepage Hero](./docs/images/homepage-hero.png)

## Documentation

Project documentation has been organized under the `docs` directory:

- [Documentation Hub](./docs/README.md)
- [Development Guide](./docs/DEVELOPMENT.md)
- [Production Operations Guide](./docs/PRODUCTION.md)

## Platform Scope

- Marketing website and conversion funnels
- Admin portal for broker, user, and enquiry operations
- Analytics surfaces for landing traffic and operational visibility
- API-driven backend workflows for admin and marketing services

## Core Value Proposition

- Structured trading support based on predefined strategy terms
- Coverage across Gold, Crypto, Forex, and Commodities
- Transparent client-first operating model through broker-account ownership

## Quick Start

```bash
npm install
npm run dev
```

For full setup, environment standards, and deployment readiness, use the documentation links above.
